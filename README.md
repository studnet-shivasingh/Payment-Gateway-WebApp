# Payment Gateway Web Application

A full-stack payment gateway web application developed for secure online transaction processing and real-time payment handling. The project integrates Razorpay Payment Gateway APIs with a responsive frontend and backend architecture to simulate a modern online payment system.

The application allows users to initiate payments securely while maintaining smooth transaction flow, backend validation, and database integration.

---

# Features

* Secure online payment integration
* Real-time transaction handling
* Responsive user interface
* Backend API integration
* Payment request management
* MongoDB database connectivity
* Error handling and API response management
* Lightweight and scalable architecture

# Installation & Setup

1. Clone the repository

git clone link 

2. Install frontend dependencies

npm install

3. Install backend dependencies

npm install

4. Create .env file and add:

RAZORPAY_KEY_ID=
RAZORPAY_SECRET=
MONGO_URI=

5. Start frontend server

npm run dev

6. Start backend server

npm start
